# 📰 AI 信息阅览

每日自动聚合全球新闻、AI 动态与跑圈资讯的智能阅读平台。

## ✨ 功能介绍

| 功能 | 说明 |
|------|------|
| **📰 时事新闻** | 国际新闻 + 中国新闻，每日 RSS 自动抓取 |
| **🤖 AI 动态** | AI 全球动态 + AI 中国动态，跟踪最新科技进展 |
| **🏃 跑圈** | 中国跑圈 + 国际跑圈 + 运动健康知识 |
| **🇨🇳 中文翻译** | 英文文章一键翻译为中文（DeepSeek 驱动） |
| **⭐ 收藏** | 收藏感兴趣的文章，上限 20,000 条 |
| **📋 浏览历史** | 自动记录已读文章，上限 200 条 |
| **🔍 搜索** | 按关键字或日期搜索历史文章 |
| **📄 分页** | 每页 20 条，支持上下页和页码跳转 |
| **📌 今日标记** | 清晰区分今日新消息与历史消息 |
| **⏰ 自动更新** | 每日凌晨 4:30 自动抓取最新资讯 |

## 🚀 快速使用

打开网址即可阅读，无需注册登录。

- 点击顶部导航栏切换板块
- 点击「🇨🇳 翻译」翻译英文文章
- 点击「☆」收藏感兴趣的内容
- 使用搜索栏按关键字或日期查找

## 🛠️ 技术栈

- **后端**: Node.js + Express
- **前端**: 原生 HTML/CSS/JavaScript
- **翻译**: DeepSeek API
- **数据**: RSS 源抓取 + JSON 文件存储
- **部署**: Railway
